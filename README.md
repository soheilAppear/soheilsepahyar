# soheilsepahyar.github.io

Personal website for Soheil Sepahyar (GitHub Pages).

## What is included
- `index.html` and `styles.css` (modern, clean layout)
- `Soheil_Sepahyar.jpg` (profile photo used in the hero and navbar)
- `resume.pdf` (linked as CV)

## How to publish
1. Replace the files in your GitHub repo root with the contents of this zip.
2. Commit and push to `main`.
3. GitHub Pages will publish it at:
   https://soheilappear.github.io/soheilsepahyar/

## Quick edits
- Update text in `index.html` under sections: About, Current work, Experience, Publications, Teaching
- Replace `resume.pdf` anytime with your latest CV (keep the filename)
- Replace `Soheil_Sepahyar.jpg` anytime (keep the filename)
